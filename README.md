# GoUtils

[![CodeCov](https://codecov.io/gh/MegaShow/goutils/graph/badge.svg?token=VI2BCE8X5H)](https://codecov.io/gh/MegaShow/goutils)

`go.icytown.com/utils` 是一个 MegaShow 的 Golang 工具库，基于 Go 1.21 和泛型能力，实现了常用的数据结构和工具函数。

## 安装

```sh
go get go.icytown.com/utils
```
